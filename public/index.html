About

Enigma PRO is a web application designed allow you encrypt and decrypt your messages efficiently!

API

You can get an encrypted or decrypted message making the following request:

            async function getEnc() {

                const data = {
                    text: 'Message to be encrypted!',
                    encrypt: true
                }
        
                let ft = await fetch('/api', {
                    method: 'POST',
                    headers: {
                        'Content-Type': 'application/json'
                    },
            
                    body: JSON.stringify(data)
                });
            
                let res = await ft.json();
                
                if(data.encrypt) {
                    console.log(res.result);
                } else {
                    console.log(res.result.split('^').join(' '));
                }
            }
        
            getEnc();

Explanation

In the variable data: the propety text is the text to be encrypted or decrypted, the boolean encrypt specifies either the text have to be encrypted or decrypted, true means that the text have to be encrypted and false means that the text have to be decrypted.
